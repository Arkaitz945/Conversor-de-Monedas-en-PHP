💱 Conversor de Monedas en PHP
Proyecto personal de aprendizaje para practicar PHP, CSS y JavaScript. Puede contener errores o cosas mejorables, está hecho con ganas de volver a programar más que con afán de perfección.
¿Qué hace?
Permite convertir entre distintas divisas (USD, EUR, GBP, JPY) utilizando tasas de cambio reales obtenidas a través de una API externa. También guarda un historial de las últimas conversiones en el navegador.
Funcionalidades

Conversión entre divisas en tiempo real
Botón para intercambiar moneda de origen y destino
Historial de las últimas 5 conversiones (guardado en localStorage)
Validación del formulario antes de enviar
Animación suave al mostrar el resultado

Tecnologías usadas

PHP (sin framework)
HTML y CSS
JavaScript (vanilla)
API externa para las tasas de cambio

Cómo arrancarlo en local
Necesitas tener PHP instalado. Luego desde la raíz del proyecto:
bashphp -S localhost:8000
Y abre el navegador en http://localhost:8000.

Aviso
Este proyecto está hecho con fines de aprendizaje. No está pensado para uso en producción y seguramente tiene más de un fallo. Cualquier sugerencia es bienvenida.
